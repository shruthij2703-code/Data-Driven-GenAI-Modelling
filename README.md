## ğŸ”¹ Random Forest Example

### ğŸ¯ Use Case: Spam Email Detection

The model classifies emails as **SPAM or NOT SPAM** using:

* Presence of keywords (e.g., "FREE")

* Reduces overfitting
* More accurate than a single decision tree
